# ✈️ Pathfinder – Travel Booking Web Application

A large-scale travel booking web application built with Next.js and TypeScript, designed to simulate a complete platform for discovering and reserving stays, experiences, cars, and flights.

---

## Overview

Pathfinder is a comprehensive booking platform that covers multiple travel services within a unified user experience.

The project focuses on building a scalable frontend architecture, handling complex UI flows and delivering a rich, modern interface for travel exploration and reservations.

It was developed as part of an advanced learning journey, with an emphasis on real-world application structure and user experience design.

---

## Core Features

### 🌍 Exploration & Discovery
- Browse stays, experiences cars, and real estate
- Advanced filtering system
- Detailed item pages with descriptions, ratings, and reviews

### 📅 Booking Experience
- Reservation flow UI
- Availability and pricing presentation
- Wishlist / save functionality

### 👤 User Account
- Profile management
- Saved listings
- Booking history UI
- Account settings (password, billing, personal info)
- Notifications system
- Logout functionality

### 🏠 Hosting & Listings
- List properties and services
- Author profiles with reviews and ratings

### 🌐 Global Experience
- Language selection
- Currency selection

### 🎨 UI/UX Enhancements
- Dark / Light mode
- Smooth animations with Framer Motion
- Modern responsive layout

### 📰 Additional Pages
- Blog
- About
- Contact
- Help
